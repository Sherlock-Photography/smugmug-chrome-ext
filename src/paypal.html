Add PayPal buttons to this SmugMug gallery

Step one: Create a Buy Now button on PayPal

On PayPal's website, log in to your account. Click on the "Merchant Services" tab, then click on the "Buy Now buttons" link on the right.

In Step 1, change the "button type" to "Buy Now". Leave the Item Name blank.

If you are offering multiple photo sizes or multiple product options, tick the "add drop-down menu option" and fill in the details you want displayed.

Enter your shipping and tax information. Choose the option to use your secure merchant account ID.

In Step 2, untick the option to "save button at PayPal".

Now click "Create Button". On the next screen that appears, click the link "Remove code protection".

Finally, copy the code from the code box and paste it into the box below.

Step two: Select the photos to modify

Click on any image to select it, click it again to deselect it.

Select multiple images at the same time by holding down shift.

Step three: Go!

This will add PayPal buttons to the captions of each of the photos you have selected. This will overwrite any PayPal button that is currently in use on those photos.
